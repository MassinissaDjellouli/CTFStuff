# Mobile

## Android
- apkeasytool to extract files from apk
- apktool to decompile
  - apktool d [file.apk] -f (--force) -r (--no-res)
- to rebuild:
  - apktool b -f -r [folder] -o [output.apk] 
- Genymotion to emulate android
- APKSigner to sign
